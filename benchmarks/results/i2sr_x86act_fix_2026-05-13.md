# I2_SR x86 ACT Packing Fix, 2026-05-13

This note records the fix for the first `I2_SR` quality failure.

## Root Cause

The direct packed writer was emitting a row-group-of-four layout. The active
x86 BitNet runtime in this fork defines `ACT_PARALLEL` through
`include/gemm-config.h`, and the validated `llama-quantize` path emits the
chunked x86 layout instead: each flat row-major group of 128 ternary codes is
packed into 32 bytes.

Byte-level check on `blk.0.attn_q.weight`:

| layout | code SHA-256 prefix | first 16 code bytes | matches known-good row-scale prototype |
| --- | --- | --- | --- |
| old direct row-group pack | `a54eacd6eb03` | `911a405628458198661686858606a291` | no |
| x86 ACT chunk-128 pack | `efe477cf31e2` | `820a4956046984896a25868aa0169084` | yes |

After the writer fix, checked `I2_SR` tensors have byte-identical ternary code
payloads to the known-good row-scale `I2_S` prototype. Row-scale bytes differ
slightly because direct export preserves the trained FP32 row scales, while the
prototype quantizer recomputed row maxima from the intermediate F16
materialization.

## Fixed Result

| artifact | file MiB | PPL | PPL tok/s | prompt tok/s | decode tok/s | status |
| --- | ---: | ---: | ---: | ---: | ---: | --- |
| old `I2_SR` row-group direct pack | `1211.3` | `20,074,699.9423` | `149.90` | `212.10` | `19.01` | failed |
| fixed `I2_SR` x86 ACT pack | `1211.3` | `38.8477` | `150.34` | `211.67` | `19.07` | quality preserved |
| older row-scale `I2_S` prototype | `1211.3` | `38.8832` | `153.52` | `218.17` | `18.97` | quality preserved |

The fixed `I2_SR` result is now quality-equivalent to the row-scale prototype
on the fixed WikiText excerpt while keeping the cleaner separate qtype/file
type contract.

## Remaining Limit

This is still a downstream candidate path. The source tree does not keep
`GGML_TYPE_I2_SR` active by default; the runtime patch must be applied before
loading the fixed artifact. Productization still requires promoting the qtype
into the maintained runtime contract and adding regression tests for scalar
`I2_S` and row-scale `I2_SR` packing.
